Esta atividade foi desenvolvida durante o curso de Java para back-end no programa Lets Code from ADA em parceria com as lojas Americanas.
Este projeto simula um gerenciador de arquivos e pastas em que é possível, ao passar um endereço a ser trabalhado, criar pastas, lista-las,
deleta-las, criar arquivos para pastas específicas e deletar estes arquivos.

Esta atividade foi desenvoldia para prática de noções de uso de interface, responsabilidade e abstração.
